# 지방흡입 (Liposuction)

> 출처: PRS, Aesthetic Surgery Journal, ASPS(미국성형외과학회), 대한성형외과학회, PubMed 체계적 리뷰 종합

---

## 1. 지방흡입 기법 비교

### 1-1. PAL (Power-Assisted Liposuction)
- **원리**: 전동 캐뉼러가 진동하며 지방 파쇄·흡입
- **장점**: 의사 피로 감소, 고밀도 지방(등·옆구리) 효과적
- **단점**: 기존 흡입법 대비 큰 차별점 제한적

### 1-2. VASER (Vibration Amplification of Sound Energy at Resonance)
- **원리**: 초음파 에너지로 지방세포만 선택적 유화(emulsification)
- **장점**: 혈관·신경 보존, 피부 수축 효과, 정밀 조각(Hi-Def) 가능
- **적용**: 복근 라인 조각(abdominal etching), 남성 가슴(여유증)
- **합병증**: 열 손상(thermal injury) 주의, 숙련도 필요

### 1-3. LAL (Laser-Assisted Liposuction)
- **원리**: 레이저 에너지로 지방 용해 + 피부 수축
- **브랜드**: SmartLipo, SlimLipo
- **장점**: 피부 타이트닝 효과
- **단점**: 열 화상 위험, VASER 대비 장점 제한적

### 1-4. Body-Jet (Water-Assisted Liposuction, WAL)
- **원리**: 물줄기로 지방 분리 → 흡입. 지방세포 생존율 높음
- **장점**: 자가지방 이식용 채취에 적합, 조직 손상 적음
- **단점**: 대량 흡입에는 시간 소요

### 1-5. 전통적 흡입(SAL, Suction-Assisted Liposuction)
- 가장 기본적인 방식. 캐뉼러 + 음압으로 흡입
- 비용 합리적, 대량 흡입에 적합

## 2. 부위별 지방흡입

### 2-1. 복부
- **가장 많은 시술 부위**. 상복부, 하복부, 옆구리(러브핸들) 구분
- 360도 흡입(circumferential): 복부 전체 + 허리 라인 동시
- 주의: 복벽 관통(perforation) 위험 → 캐뉼러 각도·깊이 주의

### 2-2. 허벅지
- 안쪽(inner), 바깥(outer), 앞(anterior), 뒤(posterior) 구분
- 안쪽 허벅지: 피부 처짐 동반 가능 → 리프팅 병행 고려
- 바깥 허벅지(승마살): 여성 인기 부위

### 2-3. 팔
- 상완 안쪽. 피부 탄력에 따라 추가 절제 필요할 수 있음
- VASER 선호: 피부 수축 효과

### 2-4. 턱밑/이중턱
- 소량 흡입. 턱끝·목 라인 개선
- 리프팅 병행(submental liposuction + platysmaplasty)

### 2-5. 등/브라라인
- 고밀도 지방. PAL/VASER 효과적
- 여성 브라 라인 자신감 개선

## 3. 합병증

| 합병증 | 빈도 | 설명 |
|--------|------|------|
| 부기·멍 | 100% | 정상. 2~6주 소실 |
| 불균일(contour irregularity) | 5~20% | 가장 흔한 미용적 합병증 |
| 장액종(seroma) | 2~5% | 체액 고임. 흡인 필요 |
| 감각 저하 | 일시적 | 피부 감각 둔화. 3~6개월 회복 |
| 피부 처짐 | 가변적 | 피부 탄력 부족 시 |
| 혈전/폐색전증 | <0.1% | 대량 흡입+전신마취 시 위험 증가 |
| 지방색전증(fat embolism) | 매우 드묾 | 생명 위협. 대량 흡입 시 주의 |
| 장기 손상 | 매우 드묾 | 복벽 관통. 숙련도 관련 |

### 안전 가이드라인 (학회 권고)
- **1회 최대 흡입량**: 5,000mL (ASPS 권고). 초과 시 합병증 급증
- **대량 흡입(>5L) 시**: 입원 관찰, 수액 보충, 혈전 예방
- **BMI 35 이상**: 합병증 위험 증가. 체중 감량 후 권고

## 4. 자가지방 이식 (Fat Grafting/Transfer)

### 4-1. 얼굴 자가지방
- **부위**: 이마, 관자, 볼, 팔자, 턱끝, 눈밑
- **생착률**: 40~70%. 2~3회 시술 권장
- **합병증**: 불균일, 감염, 지방 괴사

### 4-2. 가슴 자가지방
- 1컵 이상 증대 어려움. 보형물 보조용
- 생착률 50~70%. 유방 검진 시 석회화 감별 필요

### 4-3. 엉덩이 자가지방 (BBL, Brazilian Butt Lift)
- **가장 위험한 미용 수술** (ASJ 2020): 사망률 약 1/3,000
  - 원인: 지방이 둔근 근육 아래 주입 → 정맥 유입 → 지방색전증
- **안전 권고**: 근육 위(subcutaneous)에만 주입. 깊은 주입 금지
- 최근 학회: 안전 프로토콜 강화

## 5. 압박복 (Compression Garment)

- **착용 기간**: 4~6주 (24시간), 이후 2~4주 (낮에만)
- **역할**: 부기 감소, 피부 밀착, 장액종 예방, 윤곽 형성
- **주의**: 너무 조이면 혈류 장애, 피부 괴사 가능

## 6. 회복 타임라인

| 시기 | 상태 |
|------|------|
| 수술 직후 | 압박복 착용, 드레인 (2~3일) |
| 1주 | 가벼운 일상활동. 통증 감소 |
| 2~3주 | 부기 50% 감소. 가벼운 운동 |
| 1개월 | 운동 가능(강도 서서히 증가) |
| 3개월 | 윤곽 형태 안정화 |
| 6개월 | 최종 결과 |

## 7. 최신 트렌드 (2024-2025)

- **VASER Hi-Def/4D**: 복근·가슴근 라인 조각. 남성 인기
- **Renuvion(J-Plasma)**: 흡입 후 피부 수축 장비. 헬륨 플라즈마
- **미세 캐뉼러(micro-cannula)**: 정밀 흡입, 불균일 감소
- **1일 수술(ambulatory)**: 국소마취+진정으로 입원 없이
- **3D 체형 시뮬레이션**: 수술 전 결과 예측

## 8. 재수술 주의점

- **첫 수술과의 차이**: 한 번 흡입한 부위는 조직 유착·섬유화가 생겨 캐뉼러 삽입이 어렵고, 피부와 피하층의 경계가 불균일해져 울퉁불퉁(irregularity) 발생 위험 높음
- **체크 포인트**: ① 이전 흡입 부위·기법(SAL/VASER/PAL 등) 기록 확보 ② 피부 탄력·처짐 정도 ③ 잔여 지방층 두께(초음파 측정 권장) ④ 장액종·혈종 병력 ⑤ 영양·체중 변동 안정 여부(체중 변동 심하면 결과 왜곡)
- **한계**: 완전히 균일한 결과 복원은 쉽지 않음. 울퉁불퉁은 지방이식으로 보완하는 경우도 있음
- **대체 방법 고려**: 피부 처짐 동반 시 리프팅(복부 성형·팔 거상 등) 병행 필요할 수 있음. Renuvion 같은 피부 타이트닝 장비 결합
- **경험 많은 의사 필수**: VASER·LAL 같은 열 기반 장비 사용 시 재수술 열 손상 위험 증가. 6~12개월 경과 관찰 후 검토

## 9. 체형별 추천

- **사과형(복부·옆구리 중심)**: 360도 순환 흡입(복부+옆구리+등허리) 동시. VASER로 정밀 조각도 가능
- **서양배형(엉덩이·허벅지 중심)**: 허벅지 안쪽·바깥쪽·엉덩이 아래 흡입. 피부 탄력 낮으면 리프팅 동반
- **마름형(전체적으로 마른데 특정 부위만 통통)**: Hi-Def(복근 조각)·팔 안쪽 등 정밀 부위 VASER 선호
- **근육형(남성 위주)**: VASER 4D/Hi-Def로 복근·가슴근 라인 조각. 자가지방 이식 병용 가능
- **피부 처짐 동반 체형**: 지방흡입 단독 비추천. 복부 성형(tummy tuck) 또는 팔·허벅지 거상 병행
- **자가지방 이식 목적**: Body-Jet(WAL) 권장. 지방세포 생존율 높음
- 공통: BMI 30 이상은 체중 감량 우선. 흡입 한계량(4~5L) 초과 시 합병증 위험 급증

## 10. 회복 타임라인

| 시점 | 상태 (확장) |
|------|------------|
| 1일 | 압박복 24시간, 드레인 유지, 침대 안정. 림프액 배출 많음 |
| 3일 | 부기·멍 최대치, 움직임 뻣뻣. 가벼운 산책 권장 |
| 1주 | 드레인 제거, 실밥 제거 시점. 부기 50% |
| 2주 | 일상 복귀, 가벼운 걷기·사무 업무. 압박복 24시간 유지 |
| 1개월 | 가벼운 운동 가능, 압박복 낮시간만. 부기 60~70% 감소 |
| 3개월 | 윤곽 안정화, 최종 결과 80% 수준. 림프드레나지(마사지) 권장 |
| 6개월 | 최종 결과 도달. 흉터 거의 희미, 피부 수축 완료 |

## 11. 자주 묻는 질문 (FAQ)

- **Q: 지방흡입하면 요요 없나요?** → 흡입한 부위의 지방세포 수 자체가 줄어 요요는 상대적으로 적음. 단 체중 증가 시 다른 부위에 지방이 쌓일 수 있음
- **Q: 한 번에 얼마나 뽑을 수 있나요?** → 미용 지방흡입 권장 한계 4~5L. 그 이상은 전신 상태 악화·혈역학 이상 위험 증가
- **Q: VASER랑 일반 흡입 차이?** → VASER는 초음파로 지방만 유화→흡입. 혈관·신경 손상 적고 피부 수축 효과. 대신 열 손상 위험, 비용 높음
- **Q: 압박복은 꼭 24시간?** → 수술 후 2~4주는 24시간 권장. 부기·장액종 예방, 피부 밀착에 필수. 이후 낮 시간만
- **Q: 셀룰라이트가 없어지나요?** → 지방흡입은 지방 자체는 줄이지만 셀룰라이트(피부 구조 문제) 개선은 제한적. 서브시전(Cellfina) 등 별도 시술 필요
- **Q: 재수술은 언제 가능한가요?** → 최소 6개월, 안전하게는 12개월 후. 부기 완전 소실과 조직 안정 후 평가
- **Q: BBL(엉덩이 자가지방)은 왜 위험한가요?** → 근육 아래 주입 시 정맥 유입으로 지방색전증 가능. 사망률 약 1/3,000. 근육 위(피하)에만 주입하는 안전 프로토콜 준수 필수
