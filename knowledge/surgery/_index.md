# Surgery Knowledge Index

모든 knowledge/surgery/*.md는 공개 의학정보·공공 의료포털·학회 자료 기반 요약.
상담 AI 시스템 프롬프트에 부위별로 주입되는 참고자료.

## 파일
- eye.md — 눈 (쌍꺼풀·눈매교정·트임)
- nose.md — 코 (실리콘·자가연골·코끝)
- breast.md — 가슴 (보형물·지방이식)
- contour.md — 안면윤곽 (사각턱·광대·턱끝)
- lipo.md — 지방흡입·이식

## 원칙
1. **단정·진단·가격·병원명·의사명 금지**
2. **부작용·합병증 반드시 포함** (고객 오판 방지)
3. **연령대 경향** 포함 (상담실장이 맥락 있는 추천 가능하게)
4. **"~경향", "~많아요", "~편이에요"** 완화 표현

## 참고 출처 공통
- 서울대학교병원 의학정보 (snuh.org/health/nMedInfo)
- 대한성형외과학회 (plasticsurgery.or.kr)
- 식품의약품안전처 의료기기 허가정보
- 건강보험심사평가원 통계
- ASPS, ISAPS 공개 데이터 (참고)
