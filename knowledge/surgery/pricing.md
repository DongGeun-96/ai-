# 성형·시술 대략적 비용 참고 (2025년 기준)

> **출처**: 건강보험심사평가원 비급여진료비 공개 데이터, 한국소비자원 가격조사, 대한성형외과학회 참고자료, 공개 시세 종합
> **주의**: 아래 가격은 대략적인 시세 범위이며, 실제 비용은 병원·의료진·재료·지역에 따라 크게 달라질 수 있습니다. 정확한 비용은 대면 상담 시 확인하세요.
> **라이선스**: 공공누리 제1유형(출처표시) 기반 공공데이터 + 공개 시세 종합

## 눈 (eye)

| 시술명 | 대략적 가격대 | 가격 영향 요소 |
|--------|-------------|---------------|
| 매몰법 쌍꺼풀 | 60~150만원대 | 병원 등급, 매몰 방식(2점/3점), 마취 방법 |
| 절개법 쌍꺼풀 | 100~250만원대 | 피부 절제량, 지방 제거 여부, 병원 |
| 눈매교정 (안검하수) | 80~200만원대 | 단독/쌍꺼풀 병행, 비절개/절개 |
| 앞트임 | 50~120만원대 | 범위, 병원, 복합 시 할인 |
| 뒤트임 | 50~120만원대 | 범위, 결막/외측 방식 |
| 밑트임 | 60~130만원대 | 피부 절제 여부 |
| 상안검 거상술 | 150~350만원대 | 처진 정도, 지방/근육 조절 |
| 하안검 성형 | 150~300만원대 | 지방 재배치/제거, 피부 절제 |

|--------|-------------|---------------|
