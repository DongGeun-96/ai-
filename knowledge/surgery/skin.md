# 피부 시술 지식베이스 (상담용)

공개 의학정보·식약처·피부과학회 자료 기반 요약.
⚠️ 단정·가격·특정 병원/의사 금지. "경향", "많아요", "편이에요" 톤.

---

## 1. 보톡스 (Botulinum Toxin)

### 원리
- 보툴리눔 독소를 근육에 주입 → 신경-근육 신호 차단 → 근육 이완.

### 용도
- **주름**: 미간, 이마, 눈가(까마귀발), 입꼬리, 턱끝(사과볼).
- **사각턱 축소**: 교근(masseter)에 주입 → 근육 위축 → 얼굴선 갸름.
- **다한증**: 겨드랑이·손바닥 땀 감소.
- **기타**: 종아리 축소(비복근), 어깨 승모근, 광대근.

### 효과 지속
- 3~6개월. 반복 주사 필요. 반복할수록 효과 지속이 길어지는 경향.

### 부작용
- 멍·부기(경미), 표정 어색함(과주입 시), 눈꺼풀 처짐(이마 과주입 시), 저작력 저하(사각턱).
- 드물게: 두통, 감기 유사 증상.

### 연령 경향
- **20대**: 이마·미간 잔주름 예방, 사각턱.
- **30대+**: 눈가·입꼬리·턱끝 주름 관리 + 사각턱 유지.
- **40대+**: 주름 + 리프팅 병행.

## 2. 필러 (Dermal Filler)

### 원리
- 히알루론산(HA), 칼슘하이드록시아파타이트(CaHA/래디에스), 폴리카프로락톤(엘란세) 등을 피부 아래 주입 → 볼륨·윤곽 보강.

### 용도
- **볼륨**: 볼·이마·관자놀이·턱끝·코(비수술)·팔자주름·입술.
- **눈밑**: 다크서클·꺼짐 개선(연질 HA).
- **손등**: 노화로 인한 볼륨 감소.

### 재료별 차이
| 재료 | 지속 | 특징 |
|------|------|------|
| 히알루론산(HA) | 6~18개월 | 녹일 수 있음(히알루로니다제), 가장 안전 |
| CaHA(래디에스) | 12~18개월 | 콜라겐 생성 촉진, 녹일 수 없음 |
| PCL(엘란세) | 2~4년 | 장기 지속, 녹일 수 없음 |
| PLLA(스컬트라) | 2년+ | 콜라겐 자극형. 즉각 볼륨 X, 점진적 |

### 부작용
- 멍·부기(흔함), 뭉침·결절, 비대칭, 이물감.
- **중대**: 혈관 폐색 → 피부 괴사, 시력 손상(코·이마 부위 특히 주의). 매우 드물지만 심각.
- 감염, 필러 이동.

### 연령 경향
- **20대**: 코 필러(비수술), 입술, 애교살.
- **30대**: 팔자·볼 볼륨 보완.
- **40대+**: 볼·관자놀이·손등 볼륨 + 리프팅 병행.

## 3. 레이저 시술

### 3-1. 레이저 토닝 (Low-fluence Q-switched Nd:YAG)
- **원리**: 낮은 에너지로 반복 조사 → 멜라닌 점진 파괴.
- **용도**: 기미·색소침착·피부톤 균일화.
- **횟수**: 5~10회(2주 간격).
- **부작용**: 일시적 홍조, 드물게 탈색(백반). 과다 시술 시 색소 악화.

### 3-2. IPL (Intense Pulsed Light)
- **원리**: 광범위 파장 빛 → 멜라닌·혈관 동시 타겟.
- **용도**: 잡티·모공·홍조·주근깨·피부결 전반.
- **장점**: 다운타임 적음. 일상 가능.
- **부작용**: 일시적 홍조·갈반, 화상(드묾).

### 3-3. 프락셀 (Fractional Laser)
- **원리**: 미세 레이저 점 조사 → 주변 정상 조직이 치유 → 콜라겐 재생.
- **용도**: 여드름 흉터·모공·잔주름·피부결.
- **종류**: 비박리(Non-ablative) vs 박리(Ablative/CO2). 박리형이 효과 강하지만 다운타임 길음.
- **부작용**: 홍조 3~7일, 각질 탈락, 색소침착(일시적). 박리형은 1~2주 다운타임.

### 3-4. 피코 레이저 (Pico Laser)
- **원리**: 피코초(10^-12초) 단위 짧은 펄스 → 색소 미세 분쇄.
- **용도**: 기미·색소·문신 제거·피부결·모공.
- **장점**: 기존 레이저 대비 다운타임 적고 안전성 높은 편.

### 3-5. CO2 레이저 (Carbon Dioxide Laser)
- **용도**: 점·사마귀·쥐젖 제거, 흉터.
- **다운타임**: 딱지 5~10일.

## 4. 리프팅 시술 (비수술)

### 4-1. 울쎄라 (Ultherapy / HIFU)
- **원리**: 고강도 집속 초음파(HIFU) → SMAS층(근막) 열 응고 → 콜라겐 수축·재생 → 리프팅.
- **효과**: 턱선·볼·이마·눈밑 처짐 개선. 1~3개월 걸쳐 효과 발현.
- **지속**: 6~12개월.
- **통증**: 높은 편(마취 크림·진통제 병용).
- **부작용**: 부기·홍조(1~3일), 드물게 신경 자극(일시적 저림), 화상.

### 4-2. 써마지 (Thermage / RF)
- **원리**: 고주파(RF) → 진피층 열 → 콜라겐 수축·재생 → 타이트닝.
- **효과**: 피부 탄력·처짐·잔주름. 즉각 타이트닝 + 점진적 개선.
- **지속**: 6~12개월.
- **통증**: 울쎄라보다 낮은 편.
- **부작용**: 홍조(당일), 드물게 화상·지방 위축.

### 울쎄라 vs 써마지 차이
| 구분 | 울쎄라 | 써마지 |
|------|--------|--------|
| 에너지 | 초음파(HIFU) | 고주파(RF) |
| 타겟 층 | SMAS·진피 깊은 층 | 진피·피하지방 경계 |
| 주 효과 | 리프팅 | 타이트닝 |
| 통증 | 높음 | 중간 |
| 다운타임 | 1~3일 부기 | 거의 없음 |

### 4-3. 실 리프팅 (Thread Lift)
- **원리**: 녹는 실(PDO·PCL·PLLA)을 피부 아래 삽입 → 물리적 당김 + 콜라겐 자극.
- **종류**: 코그실(돌기 있음, 리프팅), 모노실(매끈, 탄력), 스크류실(볼륨).
- **지속**: 6~18개월.
- **부작용**: 당김감·통증 1주, 감염, 실 노출·만져짐, 비대칭.

### 4-4. 슈링크 (Shurink)
- HIFU 계열, 울쎄라보다 통증 낮은 편. 가격 접근성 좋음.
- 효과·지속은 울쎄라보다 가볍다는 평가 많음.

## 5. 스킨부스터

### 원리
- 히알루론산·연어DNA(PDRN)·엑소좀·성장인자 등을 피부에 직접 주입 → 보습·재생·톤 개선.

### 대표
- **쥬베룩**: HA + PDRN. 모공·피부결·수분.
- **리쥬란**: 연어DNA(PN). 잔주름·탄력·재생.
- **볼루마·프로파일로**: HA 기반 볼륨형 스킨부스터.

### 횟수
- 3~4회(2~4주 간격) 1세트.

### 부작용
- 멍·부기(1~3일), 알레르기(드묾), 결절(주입 기술).

### 연령 경향
- **20대**: 수분·톤·모공 관리 (쥬베룩·리쥬란).
- **30대+**: 탄력 + 잔주름.
- **40대+**: 리프팅 + 스킨부스터 병행.

## 6. 기타 피부 시술

### 6-1. 여드름·흉터
- 압출·MTS(미세침)·서브시전·필러(깊은 흉터)·프락셀·CO2·TCA 크로스.

### 6-2. 제모
- 다이오드 레이저·알렉산드라이트·IPL. 4~8회.
- 부작용: 화상·색소침착(일시적).

### 6-3. 점·색소 제거
- CO2·피코·Q-스위치. 점 크기·깊이에 따라 1~3회.

### 6-4. 물광주사 (Vital Injector)
- 기계로 HA·비타민·글루타치온 등 균일 주입. 보습·톤.
- 3~5회 1세트.

## 7. 주요 합병증 / 안전 정보
- 필러 혈관 폐색: **응급 대응** (히알루로니다제 즉시 주입) 가능한 병원인지 확인 필수.
- 레이저 화상: 피부 타입·세팅 맞춤 중요.
- 리프팅 신경 손상: 울쎄라·실 리프팅에서 일시적 가능.
- 감염: 모든 주사형 시술.

## 8. 상담 Q&A 대응
- "보톡스 맞으면 얼굴 굳어?" → 과주입 아니면 자연스러움. 용량 조절이 핵심.
- "필러 영구적?" → HA는 녹음. 엘란세·스컬트라는 오래 가지만 완전 영구 아님.
- "레이저 몇 번?" → 시술별 다름. 토닝 5~10회, 프락셀 3~5회 등.
- "리프팅 수술 대체?" → 처짐 정도에 따라. 심한 처짐은 수술적 거상 필요.
- "스킨부스터 바로 효과?" → 1~2회 후부터 체감. 3~4회 세트 권장.
