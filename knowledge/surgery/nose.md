# 코 성형 (Rhinoplasty)

> 출처: PRS(Plastic and Reconstructive Surgery), Aesthetic Surgery Journal, Archives of Facial Plastic Surgery, 대한성형외과학회, 대한안면성형외과학회, PubMed 체계적 리뷰 종합

---

## 1. 코 해부학 (Nasal Anatomy)

### 구조
- **골성 구조**: 비골(nasal bone) — 콧대 상부 1/3
- **연골성 구조**:
  - 상외측 연골(upper lateral cartilage) — 콧대 중간 1/3
  - 하외측 연골(lower lateral cartilage/alar cartilage) — 코끝·콧방울
  - 비중격 연골(septal cartilage) — 코 안쪽 칸막이, 구조적 지지
- **피부**: 콧대 피부는 얇고, 코끝 피부는 두꺼운 경향 (특히 아시아인)

### 아시아 코의 특징 (학술)
- 콧대가 낮은 편 (dorsal height 부족)
- 코끝이 둥글고 볼륨감 있음 (bulbous tip)
- 비중격 연골이 작고 약한 경향
- 피부가 두꺼운 편 → 코끝 성형 결과가 서양인보다 제한적
- PRS 2020: 아시아인 코 성형에서 구조적 지지 보강이 핵심

## 2. 콧대 성형 (Dorsal Augmentation)

### 2-1. 실리콘 보형물 (Silicone Implant)
- **재질**: 고체 실리콘 (L자형/I자형/Bird형)
- **장점**: 높이 조절 용이, 제거·교체 가능, 비용 합리적
- **단점**: 이물감, 피막 구축(capsular contracture) 가능, 장기 변형(shifting)
- **합병증 발생률**: PRS 2018 메타분석 — 전체 합병증 7~12%
  - 피막 구축 3~5%, 비틀림(deviation) 2~4%, 감염 1~2%, 돌출(extrusion) <1%
- **L자형 vs I자형**: L자형은 코끝까지 연장 → 코끝 피부 압력 → 돌출 위험 높음. 최근 I자형 + 자가연골 코끝 조합 선호

### 2-2. 고어텍스 (ePTFE, Gore-Tex)
- **재질**: 확장 폴리테트라플루오로에틸렌. 다공성 → 조직 유입
- **장점**: 자연스러운 촉감, 이물감 적음, 주변 조직과 통합
- **단점**: 제거 어려움(조직 유입으로), 감염 시 대처 복잡, 비용 높음
- **합병증**: 감염 2~3%, 변형 1~2%

### 2-3. 자가 조직 (Autologous Graft)
- **비중격 연골(septal cartilage)**: 가장 이상적. 양이 제한적 (특히 아시아인)
- **귀 연골(conchal cartilage)**: 코끝에 주로 사용. 부드러운 곡선
- **늑연골(costal cartilage/rib cartilage)**: 대량 확보 가능. 재수술·심한 변형에
  - 단점: 흡수(warping) 가능, 채취 부위 흉터·통증
- **근막(fascia)·진피(dermis)**: 얇은 커버용. 단독으로 높이 확보 어려움

## 3. 코끝 성형 (Tip Plasty)

### 3-1. 비익연골 재배열 (Alar Cartilage Reshaping)
- 하외측 연골을 절개·봉합하여 코끝 모양 변경
- **기법**: Cephalic trim, Domal suture, Interdomal suture, Columellar strut
- 코끝 돌출(projection), 회전(rotation), 정의(definition) 조절

### 3-2. 연골이식 (Cartilage Grafting)
- **Shield graft**: 코끝 정의를 높이는 방패 모양 이식
- **Cap graft**: 코끝 높이를 높이는 모자 모양 이식
- **Columellar strut**: 비주(코기둥)에 지지대 역할
- **Septal extension graft (SEG)**: 비중격 연장 → 코끝 높이·길이·회전 동시 조절. 아시아 코 성형의 핵심 기법 (PRS 2019)
- **Derotation graft**: 들린 코 교정용

### 3-3. 콧볼 축소 (Alar Reduction)
- **외측 절제**: 콧볼 바깥 절개하여 폭 줄임. 흉터 주의
- **내측 절제**: 콧구멍 안쪽 절개. 미세한 변화
- **봉합 축소(alar cinch suture)**: 절개 없이 봉합으로 폭 좁힘

## 4. 접근법 (Approach)

### 4-1. 비개방 접근법 (Closed/Endonasal)
- 코 안쪽 절개만. 외부 흉터 없음
- 시야 제한 → 경험 많은 의사 필요
- 간단한 콧대 보형물 삽입에 적합

### 4-2. 개방 접근법 (Open)
- 비주(columella)에 V자 절개 + 코 안쪽 절개
- 코끝 구조를 직접 시야 확보 → 정밀한 조작 가능
- 코끝 성형·재수술·복합 수술에 적합
- 흉터: 비주에 3~5mm, 6개월 이상 지나면 거의 안 보임

## 5. 코 재수술 (Revision Rhinoplasty)

- **빈도**: 초기 코 성형의 5~15%에서 재수술 필요 (PRS 2021)
- **난이도**: 초수술보다 현저히 높음. 흉터 조직, 구축, 재료 변형
- **주요 원인**: 구축(capsular contracture), 비대칭, 보형물 변형, 높이/모양 불만족
- **재수술 재료**: 자가 늑연골 선호 (기존 비중격 사용 불가 시)
- **회복**: 초수술보다 길어질 수 있음 (부기 3~6개월)

## 6. 비수술 코 성형

### 6-1. 코 필러 (Non-surgical Rhinoplasty)
- **재질**: 히알루론산(HA) 필러 (쥬비덤/레스틸렌 등)
- **용도**: 콧대 높이기, 작은 비대칭 교정, 코끝 미세 조정
- **지속**: 6~18개월 (제품·부위에 따라)
- **장점**: 즉각 효과, 회복기간 없음, 되돌릴 수 있음(히알루로니다제)
- **위험**: 혈관 압박 → 피부 괴사·실명 위험 (매우 드물지만 심각)
  - PRS 2020: 코 필러 혈관 합병증 발생률 약 0.05%, 실명 사례 보고 있음
  - 위험 부위: 미간(glabella), 코끝(tip), 비근부(nasion)

## 7. 합병증 총정리

| 합병증 | 빈도 | 설명 |
|--------|------|------|
| 부기·멍 | 100% | 정상 반응. 70%는 2주 내 소실 |
| 비대칭 | 5~10% | 부기 빠진 후 판단. 6개월 관찰 |
| 구축(capsular contracture) | 3~8% | 실리콘 주위 피막 수축. 재수술 필요할 수 있음 |
| 감염 | 1~3% | 항생제 치료, 심하면 보형물 제거 |
| 보형물 비틀림 | 2~4% | 외부 충격·구축에 의해 |
| 보형물 돌출 | <1% | 피부 얇아짐 → 보형물 노출 |
| 코끝 흡수/변형 | 가변적 | 연골 이식 시 일부 흡수 가능 |
| 감각 저하 | 일시적 | 대부분 3~6개월 내 회복 |
| 호흡 장애 | 드묾 | 비중격 부종, 보형물 위치 문제 |

## 8. 회복 타임라인

| 시기 | 상태 |
|------|------|
| 수술 직후 | 부목(splint) 착용, 코 막힘 |
| 1주 | 부목 제거, 실밥 제거. 외출 가능 |
| 2주 | 부기 70% 감소. 일상복귀 |
| 1개월 | 부기 대부분 소실. 운동 가능 |
| 3개월 | 코끝 부기 서서히 감소 |
| 6~12개월 | 최종 결과. 코끝 피부 두꺼우면 더 오래 걸림 |

## 9. 최신 트렌드 (2024-2025)

- **I자 실리콘 + 자가연골 코끝**: L자 보형물 대신 코끝을 자가연골로 분리 성형
- **SEG(Septal Extension Graft) 활용 증가**: 코끝 높이·길이·회전 동시 제어
- **3D 프린팅 수술 가이드**: 수술 전 시뮬레이션
- **보존적 코 성형(Preservation Rhinoplasty)**: 기존 구조를 최대한 보존하며 교정. 유럽 학파에서 시작, 아시아에서도 도입 중
- **비중격 연골 최대 활용**: 자가 조직 우선 트렌드

## 10. 재수술 주의점

- **첫 수술과의 차이**: 구축(capsular contracture)·흉터·보형물 변형으로 해부학 구조가 헝클어져 난이도가 훨씬 높음. 비중격 연골는 이미 사용됐을 가능성 높음
- **체크 포인트**: ① 이전 보형물 종류·제거 여부 ② 구축 등급(Baker IⅢ이상 주의) ③ 코끝 피부 두께·훈상 ④ 비중격 잘립 여부(더 이상 채취 불가) ⑤ 놓연골·귀연골 채취 가능성
- **회복 기간**: 초수술보다 길어짐(부기 3~6개월, 최종 결과 12개월 이상)
- **재수술 시기**: 최소 초수술 후 6~12개월 경과·부기 완전 소실 후 권장. 감염·노출 을 제외
- **현실적 기대치**: 초수술보다 변화폭 제한적. 한 번에 모든 것을 바꾸려 하기보다 단계적 접근 권장

## 11. 얼굴형별 추천

- **둥근형**: 콧대 높이기 + SEG로 코끝 길이를 조금 빼서 세로·가로 비율 개선
- **긴형(장방형)**: 과도한 콧대 높이기 금물. 낮은·중간 높이 콧대 + 코끝 정리형 선호
- **각진형**: 자연스러운 콧대 라인, 코끝 둥그레게 다듬어 인상 부드럽게
- **들창코**: 코끝 하향·데로테이션 그래프트로 코끝을 내림. 비중격 연장이 동반되는 장말은 시술이 궜주로
- **매부리코**: 몽 낮은 부위만 높이지 말고 부분적 콧대 보형물 + 코끝 세움
- 공통: 얼굴 중앙선·유한의 상태를 고려. 코만 높이면 여다른 대비 어색할 수 있음

(회복 타임라인은 8장 에 위치)

## 12. 자주 묻는 질문 (FAQ)

- **Q: 실리콘이 시간 지나면 모양이 변해요?** → 필연적 변형은 야·가능성 낮은 편. 단 구축·외부 충격·피부 앙력으로 위치 변화나 비틀림은 일정 확률로 생길 수 있음
- **Q: 자가연골가 흡수돼나요?** → 부분 흡수는 자연 경과. 놓연골는 휘어짐(warping)·흡수 가능. 귀연골는 비교적 안정
- **Q: 코 필러는 안전한가요?** → 경험 의사의 경우 비교적 안전하나, 혁관 합병증(피부괴사·실명) 확률 0.05% 수준의 위험 존재. 불가역성 필러는 권장 안 됨
- **Q: 코 수술 후 안경 제한은?** → 보형물 안정을 위해 최소 4주, 경우에 따라 2~3개월 권장
- **Q: 축놓은 제거 가능한가요?** → 가능하지만 오래될수록 조직과 유착되어 제거 난이도 상승. 교체 수술로 진행 일반적
- **Q: 코 끝 힘이 처지는 느낌이에요** → 일시적 부기·감각 저하. 대부분 3~6개월 내 회복. 지속되면 보형물 압박 여부 확인
- **Q: 질증 생겨도 지장 없나요?** → 감기·비염 심한 시기에 수술 미룸. 감염 위험 높아짐
