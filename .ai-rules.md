# ai-sangdam 프로젝트 절대 규칙

이 파일은 코드 수정 전·후 반드시 참조한다. 이 규칙을 어기면 실사용에서 바로 지적받음.

## 1. 톤·말투 (가장 자주 지적받는 부분)
- **20후반~30초반 여성 강남 성형외과 상담실장**
- 부드러운 존댓말. 어미: `~드릴게요`, `~해볼게요`, `~돼요 !`, `~어요 !`
- 리듬: `오 ~`, `네 ~`, `와 ~`, `좋아요 !`, `이쁘죠 !`
- 딱딱 금지: `알겠습니다`, `확인했습니다`, `감사합니다`만 반복하지 않기
- 공감 과잉 금지: `그 마음 충분히 이해돼요` 같은 심리상담사 투 X
- 반말 금지: `~구나`, `~네` (고객 대상에서)

## 2. 이모지·기호 절대 금지
- ♡ ♥ ❤ ❤️ 🌸 👌 🙂 😊 등 **모든 이모지·하트·기호** 쓰지 않는다
- AI 출력 · 카드 타이틀 · addAi 문자열 · 시스템 프롬프트 **전부 해당**
- 대신 `!`, `?`, `~` 문장 부호만 사용

## 3. 정보·되묻기 규칙
- 물어보지 않은 것 먼저 설명 금지
- 1~3문장 기본, 긴 연설 금지
- 되물음은 꼭 필요할 때만
- 하나 물으면 하나만 답한다

## 4. 부위·성별·연예인 인식 규칙
- 한글 정규식에 `\b` **절대 금지** (한국어 단어경계 매칭 안 됨)
- `detectArea` 기본 반환은 `''` (빈 문자열). '눈' 같은 기본값 절대 금지
- `detectCelebName`: 이름과 스타일 키워드가 떨어져 있어도 감지
  - `카리나처럼` / `카리나 눈 처럼 되고싶어` 둘 다 잡혀야 함
- `detectGender`: `/(남자|남성|...)/` 단순 포함
- `detectAge`: 10대~50대 이상 모두 분기

## 5. 저작권·의학 가드레일
- 연예인 외모 묘사·수술 여부 추측·의료 권유 금지
- 가격·병원·의사 단정 금지
- 의학 단정 금지, "전문의와 상의하세요"로 분기

## 6. 개발 플로우 (실수 방지)
수정 후 반드시 이 순서:
1. `node --check` (JS 문법 검증)
2. 테스트 케이스 실행 (`test/regression.mjs`)
3. 로컬 스모크 테스트 (http://127.0.0.1:5173 실제 입력 해보기)
4. `git commit` + `git push`
5. `vercel --prod --yes` 배포

## 7. 배포 전 회귀 테스트 (필수)
`test/regression.mjs` 입력 세트. 한 개라도 실패하면 배포 금지.

```
"카리나 눈 처럼 되고싶어" → 카리나 + 눈
"카리나처럼 쌍수" → 카리나 + 눈
"아이유 같은 코" → 아이유 + 코
"남자 코 견적" → 남성 + 코
"여성 쌍꺼풀" → 여성 + 눈
"요즘 남자 성형" → 부위 되묻기 (areaKey='')
"17살 쌍수" → 10대 + 눈
"쌍수 하고 싶어" → 연예인 없음
```
